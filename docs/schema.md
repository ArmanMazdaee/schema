The schema.json file consists of a set of sections,sub-sections with related properties.
The current version of schema.json consists of the following sub-sections:

    * core
    * work
    * education
    * courses
    * volunteer
    * publications
    * legal
    * skills
    * awards
    * projects
    * certificates
    * references
    * languages
    * interests
    * meta

#core

    ## name    
    ## title
    ## image
    ## email
    ## phone
    ## url
    ## summary
    ## currentLocation
    ## permanentLocation
